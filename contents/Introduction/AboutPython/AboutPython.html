About python

Python is an easy to learn, general-purpose, object-oriented interpreted language focused in a clear and clean sintaxis. Is an ideal lenguage to create both scripts and rapid applications in many areas on most platforms. Python is also suitable as and extension lenguage for customizable applications, desktop graphical user interfaces, web/internet applications and games.

It's important to remeber that python is an interpreted lenguage when ones is planning to create no time consuming applications from the execution time point of view.

The python interpreter and the extensive standar library are open source. The interpreter is esealy extended with new functions and data types implemented in C or C++
