The first PySide app

The next code explains by itself the logic arround a Qt/PySide app:

from PySide.QtGui import QApplication, QWidget
from sys import exit, argv

if __name__ == '__main__':
    app = QApplication(argv)
    window = QWidget()
    window.setWindowTitle('First Pyside app')
    window.show()
    exit(app.exec_())
