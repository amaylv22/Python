Exercise. Censor

Instructions:

Write a function called censor that takes two strings, text and word, as input and returns the text with the word you chose replaced with asterisks.

  1. The strings that you will be given will contain no punctuation or upper case letters.
  2. The number of asterisks you put should correspond to the number of letters in the word.

Solution:

Downloable python script solution file: censor.py

def censor (text, word):
  new_line = []
  asterisks = '*' * len(word)
  for wds in text.split():
    if wds == word:
      new_line.append(asterisks)
    else:
      new_line.append(wds)
  return " ".join(new_line)
