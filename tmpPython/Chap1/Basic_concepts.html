Variables & data types

Variables

A variable is a word/identifier/object that hangs a value. The variables on python are defined dynamically, it means that it not neccesary to specify the type of data that will be store on them

The symbol "=" is the assignment operator, for example

x = 1
text = "hola"

Math Operations

Arithmetic operator

There are six principal arithmetic operator in python, these are:

  1. Addition → (+)
  2. Subtraction → (-)
  3. Multiplication → (*)
  4. Division → (/)
  5. Exponentiation → (**)
  6. Modulo → (%)

Comparators

There are six different comparators, each one generates a True or False result once been evaluated:

  1. Equal to (==)
  2. Not equal to (!=)
  3. Less than or (<)
  4. Less than or equal to (<=)
  5. Greater than (>)
  6. Greater than or equal to (>=)

Logical operators

Also known as boolean operators, there are three of them and they are used to obtain True or False results based on the conventional true tables of conjunction (&), disjunction (|) and negation (~).

  1. And (and)
  2. Or (or)
  3. Not (not)

Order of operation for logic operators

The order to evaluate the logical operator is at first not, next and, finally or.

This order can be changed by including parentheses.

Bitwise operators

Bitwise operations are operations that directly manipulate bits. And its syntaxis is:

element_1 operator elemnt_2

for all of them, exept for the not operator that is:

operator element

The six baisc bitwise operators are:

Right shift (>>)
This operator work by shifting the bits to the right by a designated number of slots. It is mathematically equivalent to floor dividing by 2, e.g.:
0b0010100 >> 3 = 0b000010 (20 >> 3 = 2)
0b0000010 >> 2 = 0b000000 (2 >> 2 = 0)
Left shift (<<)
This operator work by shifting the bits to the left by a designated number of slots. It is mathematically equivalent to multiplying by 2, e.g.:
0b000001 << 2 = 0b000100 (1 << 2 = 4)
0b000101 << 3 = 0b101000 (5 << 3 = 40)
Bitwise AND (&)
performs the bit-by-bit conjunction operation, and return the resultant number
Bitwise OR (|)
performs the bit-by-bit disjunction operation, and return the resultant number
Bitwise XOR (^)
performs the exclusive or operation (1 is either of the two incoming bits are 1 but noth both) on a bit level, and returns the result
Bitwise NOT (~)
The bitwise NOT operator just flips all of the bits in a single number.

NOTE: to print a decimal number in its binary representation use the bin() function.

You can also represent numbers in base 8 and base 16 using the oct() and hex() functions.

Aditionaly, using the second parameter of the int() function, you can specify the base of the number that you're trying to convert into an integer:

int("1010",2) -> 10

Terminal Printing

Basic printing

In Python the command print is used to show results and the content of variables in the console

The syntax looks like this:

print "Welcome to python"

when the message is a string literal. Or like this:

print var1

when the content is saved into a variable

String concatenation

The operator (+) used between two string is used to concatenate them.

print "string 1" + "string 2"
print var1 + var2

String formating with %

Another way to work with concatenated strings is using the "substitution operator" (%), that works as a variable inside of a string, e.g.

print "This is a message from %s" % (name_1)

It's posible to have as many variables as ones want, but all of the have to be separated by commas between the parentheses after the symbol %, this is:

print "Today, %s of the %s week of %s, you need to..." % (today, week_cont, month)

Python % code

We can specify the number of floating-point decimals to print by changing "%f" to "%.xf", where x is the number of decimals we want (be sure to note the . between % and x!).
