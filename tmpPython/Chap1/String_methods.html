Basic string methods

Acces by ofset

Each character in a string has a subscript or offset, which is the value that represent its position in the character chain. Using this "adress" ones can get to each character in the string individually

The number starts at 0 for the leftmost character and increses by one as you move character-by-charcter to the right

Therefor to obtain the character in the n position inside string var:

var[n]

The method also could be used with explicit literal strings:

"WORD"[2] → 'R'

Length of a string → len()

Gets the length of a string, that is, the number of charcaters in the chain, returned as integer

print len(var1)

Transform to upper case → upper()

Transform all characters in the string to its upper case representation

print var1.upper()

Slicing strings → [i:j]

If it is necessary a slice of a string could be obtained telling the interpreter the range of characters ID the have to be preserved. I.e., you can get the "slice" of s from character i to one character before j using s[i:j].

new_sliced_string = string_var[i:j]

separating the words of a string → split()

Return a list of the words of the string s. If the optional second argument sep is absent or None, the words are separated by arbitrary strings of whitespace characters (space, tab, newline, return, formfeed). If the second argument sep is present and not None, it specifies a string to be used as the word separator. The returned list will then have one more item than the number of non-overlapping occurrences of the separator in the string. If maxsplit is given, at most maxsplit number of splits occur, and the remainder of the string is returned as the final element of the list (thus, the list will have at most maxsplit+1 elements). If maxsplit is not specified or -1, then there is no limit on the number of splits (all possible splits are made).

The behavior of split on an empty string depends on the value of sep. If sep is not specified, or specified as None, the result will be an empty list. If sep is specified as any string, the result will be a list containing one element which is an empty string.

s.split(<sep<, maxsplit>>)

joining list of strings → join()

The method join() returns a string in which the string elements of sequence have been joined by str separator.

Following is the syntax for join() method

str.join(sequence)

Where sequence is a sequence of the elements to be joined.
